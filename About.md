# About Go

-  Go is compiled programming langauge

- Go is a general purpose programming lanaguage

- Go is type safe --> Every type must be known to the compiler

- Go contains a Garbage Collector, but it is very, minimalistic GC

- Go binaries are generally static binaries , go does not any separate runtime.
    - a minimal runtime is already included in the Go binary

- Go can cross compile to various target machines and archs
